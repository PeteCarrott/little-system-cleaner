Little System Cleaner
=====================

Little System Cleaner is an all in one system cleaner for Windows computers. Little System Cleaner combines the following tools:
 
 * Registry Cleaner
 * Registry Optimizer
 * Privacy Cleaner
 * Disk Cleaner
 * Duplicate Finder
 * Startup Manager
 * Uninstall Manager
